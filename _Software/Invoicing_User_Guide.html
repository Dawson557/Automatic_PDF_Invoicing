Steps to Create Invoices

  1. Export "ClientRevenueReportList.xlsx" from GoRendezVous and save to "Invoices" folder. (if a previous report exists overwrite it)
  2. Open the "Invoices" folder in File Explorer
  3. Click open the Create Invoices file
  4. In the command prompt that opens simply follow the instructions entering commands with Y or N.

*Note: All rates and names are taken from the OverviewSheet.xlsx. If adding new names make sure they match exactly those in the GoRendezVous system.
